<objective>
Fix jump offset calculation and label resolution by storing instruction length and using start-relative offsets; implement label namespace isolation to prevent cross-function collisions.

This resolves the example 009 segfault caused by incorrect branch target addresses and label collisions across functions. The fix involves:
1. Updating CodeGen::pending_labels to store (label, start_pos, instr_len) instead of (label, pos)
2. Modifying emit_inst to capture start position and compute instruction length (5 for unconditional jumps, 6 for conditional)
3. Rewriting the fixup loop to compute correct relative offsets
4. Adding function-indexed label prefixes to isolate namespaces

Purpose: Restore correct control flow execution.
Output: 009_control_flow runs to completion; no segfaults.
</objective>
