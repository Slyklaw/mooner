<objective>
Fix the while/for loop infinite loop bug caused by incorrect local mutable variable handling in loop conditions.

**Problem:** While/for loops with local mutable integer variables cause infinite loops.
- `let mut i = 0; while i < 5 { i = i + 1 }` - hangs
- `let mut total = 0; for i = 0; i < arr.length(); i = i + 1 { total = total + arr[i] }` - wrong output or hang

**Key Insight:** Array parameter tracking was fixed in Plan 09. The remaining bug is how local mutable variables are stored/loaded during loop condition evaluation.
</objective>
