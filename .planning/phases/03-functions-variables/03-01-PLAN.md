  <name>Task 1: Create WASM type section writer</name>
  <files>src/wasm/type.mbt</files>
  <action>
    Create a new module `src/wasm/type.mbt` with a `TypeSection` struct that:
    1. Collects function signatures (parameter types and result types)
    2. Provides `add_func(params: Array[WasmType], results: Array[WasmType]) -> Int` returning type index
    3. Encodes the type section according to WASM spec: form 0x60, param count, param types, result count, result types

  <name>Task 2: Implement local variable index allocation</name>
  <files>src/wasm/local.mbt</files>
  <action>
    Create a new module `src/wasm/local.mbt` with a `LocalAllocator` struct that:
    1. Tracks parameter count and local count
    2. Provides `alloc_param(t: WasmType) -> Int` and `alloc_local(t: WasmType) -> Int`
    3. Returns indices according to WASM spec: parameters first, then locals
    4. Generates local declarations for code section: count of each type
    
    Ensure allocator can be reset per function.

  <verify>Test allocator with mixed parameter and local types, verify indices are sequential</verify>
  <done>LocalAllocator correctly assigns indices and produces local declarations</done>
