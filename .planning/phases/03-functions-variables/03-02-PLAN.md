<task type="auto">
  <name>Task 2: Implement export section writer</name>
  <files>src/wasm/export.mbt</files>
  <action>
    Create a new module `src/wasm/export.mbt` with an `ExportSection` struct that:
    1. Collects export entries (name, kind, index)
    2. Provides `add_func_export(name: String, func_idx: Int)`
    3. Encodes the export section: export count, then each entry (name length, name, kind, index)
    4. Integrates with existing SectionWriter to add export section
    
    Export kind for functions is 0x00.
    Ensure names are UTF-8 encoded.
  </action>
  <verify>Create an export section with a single function export and verify binary matches spec</verify>
  <done>ExportSection can encode a function export "add" with index 0</done>
</task>

<task type="auto">
  <name>Task 3: Implement import section writer</name>
  <files>src/wasm/import.mbt</files>
  <action>
    Create a new module `src/wasm/import.mbt` with an `ImportSection` struct that:
    1. Collects import entries (module name, field name, kind, type index)
    2. Provides `add_func_import(module: String, field: String, type_idx: Int) -> Int` returning import index
    3. Encodes the import section: import count, then each entry (module length, module, field length, field, kind, type index)
    4. Integrates with existing SectionWriter to add import section
    
    Import kind for functions is 0x00.
    Ensure module and field names are UTF-8 encoded.
  </action>
  <verify>Create an import section with a single function import and verify binary matches spec</verify>
  <done>ImportSection can encode a function import "env" "add" with type index 0</done>
